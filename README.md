# Object Detection and Tracking

This project uses YOLOv8 and OpenCV for real-time object detection and tracking.

## Features
- Real-time object detection
- Object tracking with unique IDs
- Webcam support
- YOLOv8 model

## Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8

## Installation

pip install -r requirements.txt

## Run

python app.py
